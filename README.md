# TechFlow Solutions - Sistema de Gerenciamento de Tarefas

## Objetivo
Projeto acadêmico para simular o desenvolvimento de um sistema ágil utilizando GitHub,
Kanban, CRUD em Python e controle de qualidade com GitHub Actions.

## Escopo
- CRUD de tarefas
- Priorização de tarefas
- Testes automatizados
- Pipeline CI

## Tecnologias
- Python
- Flask
- Pytest
- GitHub Actions

## Como executar
```bash
pip install -r requirements.txt
python app.py
```

## Gestão de Mudanças
Foi adicionada a funcionalidade de prioridade das tarefas após solicitação do cliente.
